Terra Nova World Editor (730KB)


This is Release 1 of the Terra Nova World Editor package. It contains TN-Edit V0.70 (Terrain Editor), EdMis V0.3 (Mission/Object Editor) and tools to modify the textures and the sprites in TN.
The editor work with all three official Terra Nova releases: The two demos and of course with the full version.The system requirements are: A PC who is capable of running TN (any version) with approximatly 8M of RAM and a Vesa video card which can display a resolution of 640x480 in 16Bit colors.
Installation:Just unzip the editor package into the same directory you've installed Terra Nova into.
Notes on the installation of TN full version: You must do a maximum installation! After you've done that, enter your TN-directory and do the following: (at a dos prompt)
 mkdir maps
cd maps
copy {your cd drive name:}\maps\*.*
This installs all maps onto your HD. Normaly they wont be installed and only read from cd if needed. But in order to be able to modify them, you have to install them.
The documentation is available only online.



Terra Nova 1st Demo (7.8MB)

The first demo of Terra Nova released by Looking Glass Studios.
Download this demo, if you don't have it already, to be able to play the new Operation "Stone Henge".
Instructions: Unzip it into any directory you like. Enter that directory and call "SETUP". Finally type "TNDEMO" to play the game.
Alternate download sites: Looking Glass: TN_SFC.EXE
Instructions for alternate download: Create a new directory. Move the file "TN_SFC.EXE" there. Start it. Now call "SETUP". Finally type "TNDEMO" to play the game.
HINT: Terra Nova (Demo) is basically a DOS game! If for some reasons you run into problems under Windows, just go to DOS-Mode and try again! In DOS-Mode, you need a driver for your mouse. Normally its already installed on every computer. If not, download one from microsoft, logitech or anyone else. They all have apropriate DOS-Mouse-Drivers. For more information about installing Terra Nova (Demo) look at Looking Glass: TN FAQ
