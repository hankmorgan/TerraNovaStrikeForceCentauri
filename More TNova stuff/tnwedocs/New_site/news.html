28.10.99 Oooops. What a long time I did not update this homepage. Shame on me. But I was not completly lazy: I wrote a tool TGen to generate landscapes automatically. It has an interface to the landscape generator TerraGen, so you can generate a landscape in TerraGen and import it via TGen into Terra Nova! The basic idea of this tool is to take a given heigthfield and calculate the textures on it automatically. Here is a first screenshot of an automatically generated landscape without any manual change.
